🚀 Triển khai hệ thống E-commerce với Prisma 7 + Supabase + Render + Vercel
1. 🎯 Mục tiêu hệ thống

Xây dựng và deploy một hệ thống fullstack:

Frontend: Next.js (deploy trên Vercel)
Backend: Node.js + Express (deploy trên Render)
Database: Supabase (PostgreSQL)
ORM: Prisma v7
2. 🧠 Kiến trúc tổng thể
Frontend (Next.js - Vercel)
        ↓ API
Backend (Express - Render)
        ↓ Prisma ORM
Database (Supabase - PostgreSQL)
3. ⚠️ Các vấn đề thực tế gặp phải
❌ 1. Prisma 7 thay đổi config
Không còn dùng:
url = env("DATABASE_URL")

👉 Lỗi:

The datasource property `url` is no longer supported
❌ 2. Prisma CLI không đọc ENV

👉 Lỗi:

datasource.url is required

👉 Nguyên nhân:

thiếu .env
config sai prisma.config.ts
❌ 3. Sai connection Supabase

👉 Lỗi:

P1000 Authentication failed

👉 Nguyên nhân:

dùng sai port (6543 vs 5432)
sai password
❌ 4. Migration không chạy

👉 Lỗi:

No migration found

👉 Nguyên nhân:

chưa tạo migration local
❌ 5. Seed lỗi

👉 Lỗi:

ECONNREFUSED

👉 Nguyên nhân:

DB chưa connect
bảng chưa tồn tại
4. ✅ Giải pháp triển khai chuẩn
🧩 4.1 Prisma 7 Configuration
schema.prisma
datasource db {
  provider = "postgresql"
}

generator client {
  provider = "prisma-client-js"
}
prisma.config.ts
import "dotenv/config";
import { defineConfig } from "prisma/config";

export default defineConfig({
  schema: "prisma/schema.prisma",
  datasource: {
    url: process.env.DATABASE_URL!,
  },
});
🧩 4.2 Environment
Local
DATABASE_URL=postgresql://localhost:5432/devdb
NODE_ENV=development
Production (Supabase)
DATABASE_URL=postgresql://postgres.xxx:PASSWORD@aws-...:5432/postgres
NODE_ENV=production
🧩 4.3 Migration Flow
Local
npx prisma migrate dev --name init
Production
npx prisma migrate deploy
🧠 Insight:

❗ migrate deploy KHÔNG tạo migration
→ chỉ chạy migration đã có

🧩 4.4 Seed Strategy
Phân biệt môi trường
if (process.env.NODE_ENV === "production") {
  // seed nhẹ
} else {
  // seed full data
}
Tránh duplicate
const count = await prisma.category.count();

if (count === 0) {
  // seed
}
🧩 4.5 Scripts chuẩn
{
  "scripts": {
    "dev": "ts-node-dev src/index.ts",
    "build": "prisma generate && tsc",
    "migrate:prod": "prisma migrate deploy",
    "seed": "tsx src/scripts/seed.ts",
    "start": "node dist/index.js"
  }
}
